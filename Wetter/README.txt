Wetter Dashboard – Projektübersicht

Diese Webanwendung zeigt:
- Das aktuelle Wetter in einer eingegebenen Stadt
- Eine 3-Tage-Wettervorhersage
- Eine Live-Uhr (aktuelle Zeit)
- Dark Mode (umschaltbar)
- Responsives Design (auch für Mobilgeräte geeignet)

Technologien:
- HTML
- CSS
- JavaScript
- Öffentliche API: wttr.in (ohne API-Key)

Dateien:
- index.html – Hauptstruktur der App
- style.css – Design und Dark Mode
- script.js – Logik für Wetter, Uhr und Interaktionen

Verwendung:
1. Öffne index.html in einem Browser
2. Gib eine Stadt ein (z. B. Berlin)
3. Klicke auf „Suche“ – Wetterdaten und Vorhersage erscheinen
4. Nutze den Dark Mode Button oder sieh dir die Uhr an

Vielen Dank fürs Anschauen!
